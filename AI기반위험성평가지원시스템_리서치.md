# 건설업 AI 작업위험성평가: 날씨·지리정보 연동 상세설계 리서치

## TL;DR
- 한국 무료 공개 API만으로 날씨·지리 연동 위험성평가 시스템 구축이 충분히 가능하다. 핵심 스택은 **기상청 단기예보/초단기실황/기상특보 + 에어코리아 대기오염 + V-World(지오코딩·WMS) + 안전지도(Safemap)/국가공간정보포털 재해레이어**이며, 모두 공공데이터포털에서 자동승인으로 발급된다.
- 법규상 작업중지 기준은 정량 매핑이 가능하다. 순간풍속 15m/s(타워크레인 운전)·10m/s(설치/해체·철골)·시간당 강우 1mm(철골)·체감온도 31℃/33℃(폭염, 2025.7.17 시행 의무화)가 시스템 룰엔진의 1차 임계값이 된다.
- 권장 아키텍처는 **격자좌표 변환(WGS84↔기상청 LCC) + 폴링/캐싱 + 기상특보 이벤트 트리거 + LLM Tool-calling(Agentic RAG)** 구조다. 지리정보는 굴착(지반·지하매설물), 고소/양중(강풍·낙뢰), 사면(산사태), 수변(홍수)별로 레이어를 매핑한다.

## Key Findings

### 1. 무료 날씨 API
기상청은 공공데이터포털(data.go.kr) 게이트웨이를 통해 모든 Open API를 제공하며, 개발계정은 자동승인·일 10,000회, 운영계정은 활용사례 등록 시 일 최대 100,000회로 증액된다.
- **단기예보 조회서비스**(VilageFcstInfoService_2.0): 초단기실황(매시 관측, 매시 40분 제공), 초단기예보(6시간, 매시 30분 기준 45분 제공), 단기예보(글피까지, 02시부터 3시간 간격 일 8회). 기온(T1H), 1시간 강수량(RN1), 습도(REH), 강수형태(PTY), 낙뢰(LGT), 풍속/풍향 등 제공. 전국을 5km×5km 간격의 격자(동서 149칸 745km × 남북 253칸 1,265km), **총 37,697개**로 나누어 Lambert Conformal Conic(LCC) 투영 기반으로 읍·면·동 단위 상세 날씨를 제공하며, 비관측영역은 -99로 표시된다(출처: 기상청 동네예보 데이터, datawiki.kma.go.kr).
- **기상특보 조회서비스**(WthrWrnInfoService, 요청주소 `apis.data.go.kr/1360000/WthrWrnInfoService/getWthrWrnList`): 강풍·풍랑·호우·대설·한파·폭염·태풍·건조·황사·안개·지진해일·폭풍해일 등 12종, 178개 시·군 + 44개 해역 단위. 특보 발표 시점 갱신, 예비특보·기상정보·기상속보 포함. 작업중지 트리거의 핵심.
- **태풍정보 조회서비스**(TyphoonInfoService): 중심위치·예상경로·강도.
- **중기예보**(11일까지, 06·18시 일 2회): 선제적 공정 리스케줄링용.
- **에어코리아 대기오염정보**(한국환경공단): PM10/PM2.5/O3/NO2/CO/SO2 측정소별 실시간, 개발계정 일 500회. TM좌표 기반 근접측정소 조회 가능.
- **AWS 방재기상관측**: 전국 500여 지점 실측(기온·강수·풍속) — 단, 공공기관 한정 제공.

### 2. 글로벌 무료 대안
- **Open-Meteo**: API키 불필요, 비상업 무제한, 데이터 CC BY 4.0(상업적 재배포 허용·출처표시). KMA 포함 30+ 모델, 최대 1km 해상도, 1940년~ 과거자료. 해외현장 확장 시 1순위. 단, 대기질·UV는 별도 엔드포인트(air-quality-api).
- **OpenWeatherMap**: 무료 Access API(신용카드 불필요), 분당 60콜·월 100만콜 수준.
- **Tomorrow.io/Weatherbit**: 무료 일 500콜, 하이퍼로컬.
- 상업적 사용 시 Open-Meteo 상업플랜 또는 Visual Crossing/OWM(출처표시) 검토 필요.

### 3. 기상조건-작업중지 법규 기준 (산업안전보건기준에 관한 규칙)
- **제37조(악천후·강풍)**: 비·눈·바람 등 기상 불안정 시 작업중지. 제37조제2항(개정 2017.3.3.) 원문: "순간풍속이 초당 10미터를 초과하는 경우 타워크레인의 설치·수리·점검 또는 해체 작업을 중지하여야 하며, 순간풍속이 초당 15미터를 초과하는 경우에는 타워크레인의 운전작업을 중지하여야 한다." (운전중지 기준이 개정 전 초당 20미터 → 15미터로 강화됨)
- **제140조(폭풍 이탈방지)**: 순간풍속 30m/s 초과 우려 시 옥외 주행크레인 이탈방지장치.
- **제143조**: 순간풍속 30m/s 초과 또는 중진 이상 지진 후 옥외 양중기 점검.
- **제383조(철골작업 제한)**: ①풍속 10m/s 이상 ②강우량 시간당 1mm 이상 ③강설량 시간당 1cm 이상 시 중지.
- **이동식/크롤러크레인**: 법상 명시 없으나 KOSHA GUIDE 적용. C-69-2012(이동식 크레인 안전보건작업 지침)은 "풍속이 초당 10미터 이상인 경우 작업을 중지", C-99-2015(이동식 크레인 양중작업의 안정성 검토 지침)은 "바람이 초속 5～10미터 이상 불면 인양하중표에서 20%를 감하여야 하고, 바람이 초속 18미터 이상은 작업중지"(안전보건공단 질의회신).
- **폭염(2025.7.17 시행, 고용노동부령 제448호, 제558~562조)**: "폭염작업" = 체감온도 31℃ 이상 장소 2시간 이상 작업. 체감온도 31℃ 이상 시 냉방·통풍/작업시간조정/휴식 중 하나 이상 의무. 제560조제3항(2025.7.17 신설) 원문: "사업주는 근로자가 「기상법」 제13조의2제1항에 따른 폭염특보의 기준이 되는 체감온도 33도 이상인 작업장소에서 폭염작업을 하는 경우에는 매 2시간 이내에 20분 이상의 휴식을 주어야 한다." 위반 후 사망 시 7년 이하 징역 또는 1억원 이하 벌금(고용노동부 2025.7.15 발표). 35℃ 이상 시 14~17시 옥외작업 중지 권고, 38℃ 이상 시 매시간 15분 휴식·긴급조치 외 옥외작업 중지 권고.
- **강우 시 콘크리트 타설**(국토부 가이드라인 2024.12.30): 강우 시 원칙적 타설 금지, 기상청 날씨누리 예보로 강우조건 판단, 현장 강수량계 30분 이내 간격 실측.

#### 기상특보 발표기준 (주의보 / 경보) — 기상청 예보업무규정(2025 기준)
- **강풍**: 주의보 평균풍속 14m/s 또는 순간풍속 20m/s 이상(산지 17/25); 경보 평균 21m/s 또는 순간 26m/s 이상(산지 24/30).
- **호우**: 주의보 3시간 60mm 또는 12시간 110mm 이상; 경보 3시간 90mm 또는 12시간 180mm 이상.
- **대설**: 주의보 24시간 신적설 5cm 이상; 경보 24시간 신적설 20cm 이상(산지 30cm).
- **한파**: 주의보 아침 최저기온 전일比 10℃↓+평년比 3℃↓, 또는 -12℃ 이하 2일 지속; 경보 15℃↓+평년比 3℃↓, 또는 -15℃ 이하 2일 지속.
- **풍랑**: 주의보 풍속 14m/s 3시간 지속 또는 유의파고 3m 이상; 경보 21m/s 또는 5m 이상.
- **폭염**: 주의보 일최고체감온도 33℃ 이상 2일 지속; 경보 35℃ 이상 2일 지속(2020년부터 일최고기온→일최고체감온도 기준 전환).
- **건조**: 주의보 실효습도 35% 이하 2일 지속; 경보 25% 이하 2일 지속.

### 4. 지리정보 무료 API/데이터
- **V-World(공간정보 오픈플랫폼, 국토부/공간정보산업진흥원)**: 2D/3D 지도, WebGL, Geocoder(지오코딩 일 최대 3만건), WMS 1.3.0/WFS 1.1.0. 도시계획(15종), 문화재(문화재보호도 등 3종), 사회복지시설(6종) 등 주제도 레이어. 외부망 전용, 인증키에 도메인 등록 필요.
- **국가공간정보포털(NSDI)**: 국토지반정보 통합DB센터, 산사태정보시스템, 홍수위험지도, 문화재공간정보 등 오픈API 허브. GeoJSON/KML.
- **국토지반정보(지오인포/공공데이터포털)**: 시추공 지층정보(지층명·색상·심도) 파일데이터. 굴착·흙막이 연약지반·지하수위 평가.
- **지하공간통합지도/지하안전정보시스템(JIS)**: 지하시설물 7종(상수·하수·가스·전기·통신·송유·난방)+지하구조물 6종+지반 3종, 3D. 공개신청서 기반 제공(실시간 Open API 아님). 굴착 안전 핵심.
- **산사태위험지도(산림청 산사태정보시스템)**: 5등급, 읍면동 단위 산사태예측정보. 행안부 생활안전지도 API로도 제공.
- **홍수위험지도(환경부 floodmap.go.kr)/침수흔적도**: 침수범위·깊이.
- **안전지도(Safemap, 행안부)**: 데이터 API로 산사태/홍수 등 재해레이어.
- **지진정보(기상청)**: 발생지점·규모·진도.
- **자연재해통계지도(통계청 ndsm)**: 재해영역×인구/사업체 통계 융합.

### 5. 동적 작업환경 대응 & 유사사례
- **상시 위험성평가**(2023.5.22 고시 개정): 최초-상시 체계 도입, 월(유해위험요인 파악)·주(합동점검)·일(TBM 공유) 단위. 매일 TBM에서 위험성평가 결과 공유 시 정기·수시평가 면제. → 일일 자동 위험성평가의 법적 근거.
- **대우건설 DSC(스마트건설)**: 위치기반 안전관제(위험구역 출입관제, TBM 참여, 긴급호출), 스마트태그. 위례우남역 푸르지오 적용.
- **포스코이앤씨**: 위험작업 거부권, 자율보행로봇, 스마트안전벨트.
- **학술**: Calkins et al.(2019), *Scand J Work Environ Health* 45(6):588–599(doi:10.5271/sjweh.3814) — 2000~2012년 워싱턴주 옥외 건설노동자 산재청구 63,720건 분석 결과 "외상부상 오즈비 1.005[95% CI 1.003–1.007] per one °C change in humidex"(체감온도 1℃당 외상성 부상 확률 약 0.5% 증가, CDC/NIOSH 2020.5.21 인용). 박종철·채여라(2020), 대한지리학회지 55(4):391–408(doi:10.22776/kgs.2020.55.4.391) — "2018년 우리나라에서는 44,060명의 온열질환자와 929명(95% 신뢰구간: 739~1,143명)의 초과사망자가 발생"(질병관리본부 응급실감시체계 집계 4,495명의 약 10배). 건설업 온열질환 재해 2018~2022 79명.
- **지오펜싱**: RTLS/UWB/BLE 기반 위험구역 출입경보(Smart Fence 등) 국내 다수 적용.

### (e) 통합 아키텍처 설계안
1. **좌표 변환 계층**: 현장 주소 → V-World Geocoder → WGS84 위경도 → 기상청 LCC 격자(nx,ny) 변환(Re=6371.00877, grid=5.0km, 표준위도 30/60, 기준점 경도 126E·위도 38N). 격자-행정동 매핑 테이블 캐싱.
2. **수집 계층(폴링+이벤트)**: 단기예보는 발표주기(3시간)에 맞춰 폴링, 초단기실황은 매시 폴링, **기상특보는 짧은 주기(예 5~10분) 폴링으로 발효 감지 후 이벤트 트리거**(공공 API는 웹훅 미제공 → 폴링 기반 의사 이벤트). 호출한도 보호 위해 격자·측정소 단위 캐시(TTL=갱신주기) + Redis.
3. **룰엔진**: (b) 기준표를 결정테이블로 구현 → 작업중지/제한/주의 1차 판정.
4. **지리 컨텍스트 계층**: 현장 좌표로 재해레이어(WFS/GeoJSON) 공간질의 → 굴착·사면·수변 위험 플래그.
5. **LLM/Agent 계층(Agentic RAG)**: 날씨·지리 컨텍스트를 ① 프롬프트 주입(당일 공정+좌표+기상+지리 플래그) 또는 ② **Tool-calling**(get_forecast, get_warning, get_geohazard 툴을 에이전트가 실시간 호출)으로 결합. KOSHA KRAS/CSI 지식베이스 RAG와 결합해 공종별 위험요인·대책 생성.
6. **데이터 모델**: 작업(공종·위치·일정) × 기상조건 × 지리조건 × 위험요인 매트릭스를 관계형/그래프로 설계. 작업단위(공구)별 좌표키로 기상·지리 조인.
7. **알림/에스컬레이션**: 특보 발효 시 해당 격자/시군 매핑된 현장·작업에 자동 푸시 → 작업중지 권고 워크플로우(현장소장 승인) → TBM 일지 자동 반영. 지오펜싱(RTLS)과 연계해 위험구역 진입 시 경보.

### (f) 구현 시 주의사항 및 한계
- **격자 해상도**: 기상청 5km 격자는 국지 미기상(계곡풍·빌딩풍) 미반영 → 고위험 양중작업은 현장 풍속계 실측 병행 필수(법규도 실측 전제).
- **특보 단위 불일치**: 특보는 시·군/해역 단위, 현장은 점 단위 → 시군 매핑 시 과대·과소 경보 가능. 영향예보(폭염·한파 4단계) 보조 활용.
- **호출한도**: 개발계정 한도가 낮음(특보 1만/일, 에어코리아 500/일) → 운영계정 전환 및 캐싱 필수. 다현장 확장 시 격자 중복 제거.
- **인증·도메인**: V-World는 인증키에 도메인 등록 필요·외부망 전용. 기상청 키는 발급 후 1~2시간 지연.
- **지하/지반정보**: 실시간 Open API가 아닌 공개신청·파일 기반이 많음(JIS, 시추공) → 사전 ETL·배치 적재 필요. 시추공은 공간 보간 한계.
- **폭염 의무화 신설**: 2025.7.17 시행으로 체감온도 측정·기록(연말까지 보관) 의무 → 시스템에 온·습도 기록·보관 기능 포함 권장. 체감온도 별표13의2 산식 또는 기상청 발표 체감온도 사용. 위반 후 사망 시 7년 이하 징역·1억원 이하 벌금.
- **법적 책임**: 시스템은 "권고"이며 최종 작업중지 판단·책임은 사업주/현장소장 → Human-in-the-loop 필수, 자동중지 아닌 의사결정 지원으로 설계.
- **데이터 라이선스**: 공공누리 출처표시 준수. Open-Meteo 상업 사용 시 별도 플랜.

## Recommendations
1. **PoC 1단계(4~6주)**: 기상청 단기예보+기상특보+에어코리아 3종 API + V-World Geocoder만으로 단일현장 일일 위험성평가 자동화. (b) 기준표 룰엔진 + LLM Tool-calling으로 TBM 초안 생성. 벤치마크: 룰 판정 정확도, 특보→알림 지연 5분 이내.
2. **PoC 2단계**: 지리 재해레이어(산사태·홍수·지반) 공간질의 추가, 굴착/사면 현장 대상 확대. 벤치마크: 지오해저드 플래그의 현장 실측 부합률.
3. **확장 트리거**: 다현장(>10) 전환 시 운영계정 승인 + Redis 캐시 + 격자 중복제거. 해외현장 진출 시 Open-Meteo 전환.
4. **임계값 튜닝**: 5km 격자 한계로 양중·고소 고위험작업은 현장 풍속계 IoT 연동을 2단계에 포함. 사고·아차사고 데이터로 임계값 보정.
5. **거버넌스**: 자동 작업중지 금지, 권고+승인 워크플로우로 한정. 체감온도 기록 보관 기능 의무 반영.

## Caveats
- 호출한도·갱신주기·격자해상도 수치는 2026년 6월 기준 공공데이터포털·기상청 고시 기반이며 정책 변경 가능 → 운영 전 재확인 필요.
- 폭염 작업중지 35℃/38℃는 의무가 아닌 고용부 권고 지침이며, 법적 의무는 31℃(조치)·33℃(매 2시간 20분 휴식)까지다.
- 기상특보 발표기준 수치는 제주 재난안전대책본부 페이지(기상청 예보업무규정 미러)에서 확보했으며, 기상청 날씨누리 원페이지는 동적 렌더링으로 직접 표 확인이 제한되었다 → 운영 전 기상청 공식 페이지 재확인 권장.
- 지하매설물·시추공 정확도는 갱신주기·보간 한계가 있어 실제 굴착 전 현장탐사(GPR 등) 대체 불가.
