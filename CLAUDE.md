# Hoban EHS PoC — JHA Agent (with Details)

호반그룹 LLM/RAG 기반 지능형 작업위험성평가(JHA) 지원 에이전트 PoC.

## 하네스: Hoban JHA Agent

**목표:** 자연어로 작업 내용만 입력하면 LLM/RAG가 분류·위험요인·등급·대책을 추천하고 사용자 검토 후 ERP에 등록하는 PoC 시스템 설계·구현.

**입력 데이터:** `전사 하위공종 위험요인_20260518.xlsx` (4,469행 × 11컬럼, 대공종 20 / 중공종 254 / 세부항목 1,182 / 재해형태 21 / KRAS 5×5 매트릭스).

**트리거:** "호반 JHA", "작업위험성평가 에이전트", "EHS RAG", "ERP 연동된 위험성평가", "전사 하위공종 위험요인" 관련 작업 요청 시 `jha-orchestrator` 스킬을 사용하라. 단일 에이전트 단순 작업(예: "data-engineer에게 인덱스만 다시")은 해당 에이전트를 `Agent` 도구로 직접 호출 (model: "opus") 가능.

**제약사항:**
- ERP가 유일한 외부 I/F. 안전 DB 접근은 전용 API 또는 ETL 동기화로.
- 사내 보안 가이드라인 준수, 외부 LLM 화이트리스트 필터 강제.
- AI 추천 신뢰성 검증·튜닝 기간 필요. 사용자가 항상 수정/거절 가능해야 함.
