## 3. 위험요인 매트릭스 (HazardMatrix · 경계셀 배지 포함)

```
┌─ 위험요인 · 개선대책 ──────────────────────────────┐
│ [추락] [낙하] [협착]   ← 재해형태 탭                │
├────────────────────────────────────────────────────┤
│ ▷ 추락                                             │
│ ┌────────────────────────────────────────────────┐ │
│ │ 해체 중 부재 단부에서 작업자 추락                │ │
│ │ 강도 4 × 빈도 4    [⚠ 상(잠정)] [O(잠정)]       │ │ ← 경계셀(곱16)
│ │ ▸ 경계 셀(강도4×빈도4) — 안전관리자 판단 필요    │ │
│ │ 개선대책 (체크 시 적용)                          │ │
│ │  ☑ 안전대 부착설비 설치 후 안전대 체결           │ │
│ │  ☐ 작업발판·안전난간 선설치                      │ │
│ │  출처 [R00042] [R00128]   법령 §43               │ │ ← 클릭→인용 패널
│ └────────────────────────────────────────────────┘ │
│ ┌────────────────────────────────────────────────┐ │
│ │ 와이어 해지 중 부재 낙하                          │ │
│ │ 강도 5 × 빈도 4    [상] (빨강·확정)              │ │ ← 일반 곱20 상
│ │  ☑ 출입통제·하부 접근금지구역 설정               │ │
│ │  출처 [R00210]   법령 §20                        │ │
│ └────────────────────────────────────────────────┘ │
│ [ 5×5 매트릭스 보기 ▾ ]   (RiskMatrixVisualizer)    │
└────────────────────────────────────────────────────┘
```

5×5 시각화(펼침):
```
강도\빈도  f1   f2   f3   f4    f5
 s5(사망)  하   중   중   상    상
 s4(중상)  하   하   중  [⚠상*] 상   ← * 경계셀, 현재 마커 ●
 s3(경상)  하   하   하   중    중
 s2        하   하   하   하    중
 s1        하   하   하   하    하
 ● 현재 위험요인 위치 (강도4·빈도4)  * = 경계셀(상/중 혼합)
```
