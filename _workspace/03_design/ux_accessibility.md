# UX Accessibility & 현장 환경 가이드 — 호반 JHA Agent (Phase 3)

> 작성: frontend-engineer · 기준: WCAG 2.1 AA + 현장(햇빛·장갑·소음·오프라인).
> 핵심: 등급·경계셀 정보를 **색상만으로 전달 금지** — 텍스트/아이콘/패턴 병기.

---

## 1. WCAG 2.1 AA 핵심 체크리스트

| 항목 | 기준 | JHA 적용 |
|------|------|---------|
| 색상 대비 | 텍스트 ≥4.5:1, UI ≥3:1 | 등급 배지 텍스트/배경 대비 검증. 흰 위 주황(중)은 텍스트 #7C2D12 보정 |
| 색상 비의존 | 1.4.1 | 상/중/하 = 색 + "상/중/하" 텍스트 + 아이콘(상▲ 중◆ 하●) |
| 키보드 | 2.1.1 전체 조작 | Tab 순서: 입력→분류→위험요인→인용→확정. 인용 Enter로 패널 open |
| 포커스 표시 | 2.4.7 | 2px 윤곽 + 배경 대비. shadcn focus-visible 유지 |
| 텍스트 확대 | 1.4.4 200% | rem 기반, 확대 시 가로 스크롤·잘림 없음(단일 컬럼 fallback) |
| 이름·역할·값 | 4.1.2 | ARIA role/label 전 컴포넌트 적용(ux_components.md 참조) |
| 상태 알림 | 4.1.3 | ERP/세션 상태 `aria-live`. 인용 open announce |
| 폼 라벨 | 3.3.2 | textarea·라디오·PIN 모두 `<label>` 연결 |
| 에러 식별 | 3.3.1 | 한국어 + 다음 행동 + 담당자 연락처. "Failed" 등 코드 금지 |

## 2. 경계셀·등급 비색상 표현 (★ 안전 정보 무손실)

| 등급/상태 | 색 | 텍스트 | 아이콘/패턴 | 스크린리더 |
|----------|----|--------|------------|-----------|
| 상 | 빨강 #DC2626 | "상" | ▲ 채움 | "위험등급 상" |
| 중 | 주황 #F97316 | "중" | ◆ | "위험등급 중" |
| 하 | 초록 #16A34A | "하" | ● | "위험등급 하" |
| 상(잠정) 경계셀 | 빨강+빗금 패턴 | "상(잠정)" | ⚠ | "잠정 등급 상, 안전관리자 확정 필요" |
| O(잠정) 중점등록 | — | "O(잠정)" | ⚠ | "중점등록 잠정 O" |

- 경계셀은 **빗금/점선 테두리**로 일반 '상'과 시각 구분(색 외 패턴). `aria-live="polite"`로 "확정 필요" 안내.
- ERP 등록 버튼 비활성 시 `aria-disabled="true"` + `aria-describedby`로 사유("경계셀 등급 미확정") 연결.

## 3. 현장 환경 (햇빛·장갑·소음·오프라인)

| 환경 | 대응 |
|------|------|
| 햇빛(가독성) | 고대비 모드 토글: 배경 #000 / 텍스트 #FFF / 등급색 명도 상향. 토글 상태 localStorage 유지 |
| 장갑(터치) | 터치 타겟 ≥44px(주요 버튼 56px), 버튼 간 간격 ≥8px, 스와이프 제거(탭 only) |
| 소음(알림) | 시각 알림 우선: 토스트+색 배너+(가능 시)진동. 소리 알림은 옵션 |
| 오프라인 | PWA 캐시 + 입력 localStorage 임시 저장 + 온라인 복귀 시 동기화 배너 |
| 한 손 조작 | 모바일 주요 CTA 하단 고정(thumb zone), 풀폭 버튼 |

## 4. 모바일 반응형

| 브레이크 | 레이아웃 |
|---------|---------|
| <768 (모바일) | 단일 컬럼. 인용 패널→풀스크린 모달. CTA 하단 고정 |
| 768~1024 (태블릿) | 2컬럼(매트릭스 + 인용 사이드). 안전관리자 검토 주 사용 |
| >1024 (데스크탑) | 매트릭스 + 우측 고정 인용 패널 + progress 사이드 |

- 핵심 흐름(입력→분류→위험요인)은 **모바일 우선** 설계 후 확장.

## 5. 에러·예외 a11y 문구 (한국어 actionable)

| 상황 | 문구 | role/live |
|------|------|-----------|
| RAG no_match | "관련 표준 데이터를 찾지 못했습니다. 안전관리자에게 문의하세요." + 문의 버튼 | `role="alert"` |
| LLM 파싱 실패 | "AI 응답을 해석할 수 없습니다. 다시 시도하거나 관리자에게 문의하세요." + 원본 보기 | `role="alert"` |
| 네트워크 실패 | "연결이 불안정합니다. n초 후 자동 재시도합니다." + 수동 재시도 | `aria-live="polite"` |
| 세션 만료 | "세션이 만료되었습니다. 입력 내용은 임시 저장되었습니다." + 재로그인 | `role="alertdialog"` |
| 인용 로드 실패 | "원문을 불러오지 못했습니다. 인용 ID: R00042" + 다시 시도 (ID 항상 표시) | `aria-live="polite"` |
| ERP 등록 실패 | "ERP 등록에 실패했습니다. 다시 시도하거나 안전팀(☎)에 문의하세요." + 큐 상태 | `aria-live="assertive"` |
