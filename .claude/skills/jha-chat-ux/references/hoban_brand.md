## 2. 파생 스케일 (tint/shade 50~900)

기준색을 고정 단계에 두고 명도 파생 → `tailwind.config.ts` 팔레트로 정의.

- **brand** (Orange): `500 = #EE7500`(★공식), `600 = #C75F00`(hover/active·White 위 텍스트), 50~400 tint / 700~900 shade.
- **ink** (Dark Gray): `700 = #575553`(★공식), 본문 권장 `800 = #403E3C`, 헤딩 강조 `900 = #2B2A29`.
- **steel** (Gray): `500 = #89898A`(★공식), White 위 보조텍스트 `700 = #58585A`.
- **surface**: `DEFAULT = #FFFFFF`(카드) / `page = #FAFAFA`(배경) / `sunken = #F4F4F4` / `tint = #FFF7EE`(브랜드 옅은 강조).
- **line**: `#E6E6E6` 구분선.

### 의미(semantic) 매핑
- primary / CTA / 포커스 링 / 활성 = **brand**
- 본문·헤딩 = **ink** (White 대비 ≥9:1은 800, 헤딩 900)
- 보조텍스트·비활성·구분선·아이콘 = **steel**(+ 투명도 단계)
- 배경·서피스 = **White**; 위계는 White ↔ 옅은 Gray 틴트(`surface.page/sunken/tint`)로 구분

## 3. 접근성 규칙 (대비)

- **Orange #EE7500 on White ≈ 3.5:1** → 본문 텍스트 금지. 대형 텍스트·아이콘·**CTA 배경 + White 텍스트**·보더 용도로만. 소형 오렌지 텍스트는 `brand-600 #C75F00` 사용.
- **Dark Gray on White**: `ink-800 #403E3C` ≈ 9:1, `ink-900 #2B2A29` ≈ 13:1 (≥7:1 충족).
- **Gray 보조텍스트**: White 위 `steel-700 #58585A` ≈ 7:1 (순정 #89898A ≈ 3:1는 비텍스트·아이콘·구분선에만).
- 고대비 모드(`data-contrast="high"`) 토큰 유지: 배경 흑/텍스트 백/포커스 옐로(#FDE047).

## 4. 안전 등급·경보 색상 (법적 의미 — 불변)

KRAS 위험등급(`lib/tokens.ts`): 상 `#DC2626` / 중 `#F97316` / 하 `#16A34A`.
동적 경보(INFO/WARN/STOP/EVAC), 에러/성공 시맨틱 컬러도 불변.

브랜드 오렌지(#EE7500)와 위험 "중"(#F97316) 혼동 방지:
- 브랜드 오렌지는 채도 높고 진함, CTA·로고·어시스턴트 강조에만.
- 위험 주황은 등급 배지에만, **항상 "중" 텍스트 라벨 병행** 유지.
- 둘이 한 화면에 인접하지 않게 배치.
